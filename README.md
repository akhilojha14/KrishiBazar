# 🌾 KrishiBazar

A full-stack MERN application that connects farmers directly with buyers, eliminating middlemen and ensuring fair pricing.

## 🔗 Live Demo
https://krishi-bazar-flame.vercel.app

## 💻 GitHub Repository
https://github.com/akhilojha14/KrishiBazar

## 🚀 Features
- Farmer & Buyer Authentication
- Crop Listing & Management
- Direct Farmer-to-Buyer Marketplace
- Product Search & Filtering
- Responsive UI
- Secure Login & Registration
- Order Management

## 🛠 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
